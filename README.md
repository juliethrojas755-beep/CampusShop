##  Descripción

**CampusShop** es una tienda virtual desarrollada con **HTML y CSS**, diseñada para simular una aplicación móvil real sin usar JavaScript.

Permite recorrer todo el flujo de compra de un e-commerce de manera visual e interactiva.

---

##  Funcionalidades

✔️ Navegación entre páginas
✔️ Catálogo por categorías
✔️ Filtros con CSS (`:target`)
✔️ Vista de producto
✔️ Carrito de compras
✔️ Checkout con formulario
✔️ Historial de pedidos
✔️ Perfil de usuario
✔️ Diseño responsive

---

##  Cómo funciona

El proyecto no usa JavaScript.
Toda la interacción se logra con:

* Enlaces (`<a>`)
* IDs (`#blusas`, `#pantalones`)
* CSS `:target`

---

##  Ejemplo

```css
.categoria {
  display: none;
}

.categoria:target {
  display: block;
}
```
